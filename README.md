# ncstack
ncstack is a file organizing utility for books.

## Demo
![](demo.gif)

## Dependencies
* dialog, sed, awk, grep
* zsh
* zathura

## Setup
1. Git clone this repo
2. Empty the content of text files in `texts` directory
3. Edit `ncstack`'s default directory for adding books
4. Run the script
5. Add some books
6. Enjoy, then!
